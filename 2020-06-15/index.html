今天看到通知，我的個人域名 chenpingling.com 要到期了，把這個費用繳了，再續一年。 也同時檢查原本所用的云服務器，當時調整了就沒有再部署應用，先退租。 另外一台云服務器，當時用 CentOS8 安裝了 MSSQL2019 正常運行測試了 XL 項目，階段性功能完成了，也先退租。

最近主力在理解 .NET，目前覺得很適合後續項目的進行。有些要記心得筆記，總是要找個地方放。 想到 github，為了單純和安全起見，啟用了之前註冊但沒有怎麼在用的帳號，清掉沒打理的項目，先把 Blog 的地方先建立起來。 基本上是可行的，想說再回味複習最基本的 HTML，因此先用靜態頁面的方式進行。

有試著用域名指向，好像啟用 https 需要較長的時間。決定回歸到基準點，先用github 給的二級域名，就是 chenpingling.github.io。 在內地，有些路由原因，對這種兩級域名有時候會解析不到，基本上換個DNS，像是 114.114.114.114 或是 8.8.8.8 即可。

由於是另一個 github 帳號，在操作時會和原用的帳號沖突，在不願改變預設的帳號的前提之下，可以只動當前的， 

  $ git config credential.username "chenpingling"
